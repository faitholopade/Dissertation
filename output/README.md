# Pipeline Outputs

This directory contains all outputs generated by the 13-step annotation pipeline.
Files are regenerated each time the pipeline is executed.

## Annotation Tables

| File | Step | Description |
|------|------|-------------|
| `master_annotation_table_llm_v2.csv` | 02 | LLM-annotated records |
| `master_annotation_table_final.csv` | 01 | Final merged annotation table (150 records) |
| `master_annotation_table_causal.csv` | 11 | Extended with root cause and mitigation fields |

## Evaluation

| File | Step | Description |
|------|------|-------------|

| `gold_evaluation_summary.csv` | 04 | Aggregate evaluation statistics |
| `gold_confusion_matrices.txt` | 04 | Confusion matrices for each annotation dimension |
