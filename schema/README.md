# Semantic Schema

Defines the FRIA risk record ontology used to represent annotated AI incidents
as linked data. Generated by Step 05 of the pipeline.

## Files

| File | Description |
|------|-------------|
| `fria_risk_schema.ttl` | Ontology in Turtle format |
| `fria_risk_schema.jsonld` | Ontology in JSON-LD format |
| `example_risk_record.jsonld` | Example record conforming to the schema |
| `crosswalk_glossary.json` | Term mappings to external vocabularies (DPV, VAIR, AIRO) |
| `schema_documentation.md` | Detailed design documentation |

## External Vocabulary Alignment

| Vocabulary | Purpose | URI |
|------------|---------|-----|
| DPV | Data Privacy Vocabulary | https://w3id.org/dpv/ |
| DPV Risk | Risk and harm categories | https://w3id.org/dpv/risk/ |
| VAIR | Vocabulary of AI Risks | https://w3id.org/vair/ |
| EU Rights | EU Charter fundamental rights | https://w3id.org/dpv/legal/eu/rights/ |
| AIRO | AI Risk Ontology | https://w3id.org/airo/ |
| AI Act | EU AI Act concepts | https://w3id.org/dpv/legal/eu/aiact/ |
